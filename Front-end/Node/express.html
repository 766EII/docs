# 什么是 express......?

基于Node.js平台，快速、开放、极简的Web开发框架

简单理解：express的作用和Node.js内置http模块类似，是专门用来创建Web服务器的

本质：就是一个npm上的第三方包，提供了快速创建Web服务器的快捷方法

# 安装与创建

在项目所处的目录中，运行如下的终端命令

npm i express@4.17.1
1

创建基本的web服务器

// 导入 express 模块
const express = require('express')

// 创建Web服务器
// 这里app就是服务器了
const app = express()

// ..... 中间写一些请求、定义中间件、路由

// 调用api.listen（接口号，启动成功后的回调函数)，启动服务器
app.listen(80,()=>{
    console.log('express server running at http://127.0.0.1')
})

# 监听get和post请求

// 监听 get 请求
// 参数一：请求地址
// 参数二：请求监听到后触发的函数，然后res.send() 响应回客户端
app.get('/user',function(req,res){
    res.send({"name":"zs","age":"19"})
})

// 监听post请求
app.post('/user',(req,res=>{
    res.send('请求成功')
}))
